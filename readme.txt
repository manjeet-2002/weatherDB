The program:
 -Reads yearly weather data (raw, unsorted, unfiltered) from a cvs file.
 -Adds the data to vector of userdefined type (YearAvg) to store the values for temperature, rainfall and year.
 -Removes duplicates from the data using set.
 -Sorts the data
 -Stores the data in a separate cvs file.
 -Creates Map of the data with key as year So that the weather data for a given year can be accessed from the database.
